# Spam Mail Prediction

## Project Description
This project is a Machine Learning model that classifies emails as Spam or Not Spam.

## Technologies Used
- Python
- NumPy
- Pandas
- Scikit-learn
- Jupyter Notebook

## Dataset
- mail_data.csv

## Features
- Data preprocessing
- Feature extraction using TF-IDF
- Machine Learning model for spam detection
- Predicts whether an email is Spam or Ham
